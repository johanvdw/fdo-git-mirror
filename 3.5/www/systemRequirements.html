FDO Open Source- System Requirements

This page lists the software packages required to get, build, and operate FDO on Windows XP and on Linux x86.

Packages

Package OS Requirement
bison 1.875/sed 4.1.4 both Regenerate filter and expression parser if grammar changes
doxygen 1.4.6/graphviz 2.8 both Generate API reference documentation, if desired
HTML Help 1.4 SDK XP Generate API reference documentation in .chm format, if desired
.NET Framework version 2.0 XP Build
Visual Studio 2005 XP Build
autoconf, automake, make Linux Build
Subversion Both Update source
ArcSDE 9.1 client Both ArcSDE provider
MySQL 5.0 Both MySQL provider
