Setup:

In order to execute this sample, copy the following files to the FDO SDK directory
containing ODBCProvider.dll

    Cities.xml

In addition, create an Microsoft Access ODBC DataSource with the name 'Cities' and
point it to the Cities.mdb file located in this directory.
