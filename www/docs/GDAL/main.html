Introduction

The Feature Data Objects (FDO) API provides access to data in a data store. A provider is a specific implementation of the FDO API that provides access to data in a particular data store. The FDO Provider for GDAL provides FDO with access to an GDAL-based data store.

The FDO Provider for GDAL uses the GDAL library for reading a variety of raster formats. For more information, see The Essential FDO (FET_TheEssentialFDO.pdf), the FDO Developer's Guide (FDG_FDODevGuide.pdf) and the GDAL web site (www.gdal.org).
