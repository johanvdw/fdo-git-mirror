SpatialGeometryValidity
enum SpatialGeometryValidity

SpatialGeometryValidity is an enumeration to denote the validity of a Geometry during a test of type and dimensionality..

()

Enumerator:
SpatialGeometryValidity_None  Indicates no specific validity; used for unknown or default states.
SpatialGeometryValidity_Valid  Geometry passes a test.
SpatialGeometryValidity_Invalid  Geometry fails a test.
SpatialGeometryValidity_InvalidButCanBeApproximated  Geometry fails a test, but could be approximated using allowed types.
SpatialGeometryValidity_InvalidDueToDimensionality  Geometry fails a test due to having an disallowed dimensionality.
