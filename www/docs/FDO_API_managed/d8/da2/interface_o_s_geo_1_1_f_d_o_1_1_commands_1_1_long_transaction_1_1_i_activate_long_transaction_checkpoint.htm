Detailed Description

The IActivateLongTransactionCheckpoint interface defines the ActivateLongTransactionCheckpoint command, which allows the user to activate the named checkpoint for the given long transaction. As a result, the long transaction that owns the checkpoint will be activated as well if it is not already the active long transaction. The user requires the access privilege on the long transaction for which the attempt is made to activate a checkpoint.
