Detailed Description

XmlSkipHandler can be used to skip an XML element, when reading an XML document via XmlReader. It is a XmlSaxHandler with no callback implementations. Therefore, if another XmlSaxHandler's XmlStartElement callback returns a XmlSkipElementHandler then there are no more SAX events until the end of the current element is reached.
