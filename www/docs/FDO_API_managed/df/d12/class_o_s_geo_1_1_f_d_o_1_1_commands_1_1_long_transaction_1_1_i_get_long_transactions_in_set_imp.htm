Detailed Description

The IGetLongTransactionsInSetImp class is a concrete implementation of IGetLongTransactionsInSet. The IGetLongTransactionsInSet interface defines the GetLongTransactionsInSet command, which allows the user to enumerate the list of versions participating in the long transaction selection set.
