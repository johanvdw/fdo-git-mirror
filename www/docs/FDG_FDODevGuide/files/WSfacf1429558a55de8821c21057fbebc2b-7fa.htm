Double
 
 
 

Floating point numbers have a decimal point, can be signed (-), and include an optional exponent (e{[0-9]}).

NoteIf an integer is out of the 32-bit precision range, it is converted to floating point.

Examples: 

-3.4
12345678901234567
1.2e13
