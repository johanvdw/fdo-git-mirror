Building FDO Open Source
Repository Tool
 
 

You must install a Subversion client to get and update the Fdo source code. To browse the list of available clients, click the Developer tools link at the bottom of the https://www.osgeo.org page. Pick a client and follow the instructions there to obtain and install the client. The instructions in this document are based on the use of the SVN command-line client.

For the purpose of this document, the installer svn-1.3.1-setup.exe from http://subversion.tigris.org/servlets/ProjectDocumentList?folderID=91 was run.
