Building FDO Open Source
Gzipped Tarfile
 
 

Download the FDO gzipped tar file from https://fdo.osgeo.org/downloads.html to the build directory. The name of the build directory used in this document is /home/OpenSource. The filename has the format <component>-<version>_<build>.tar.gz

The version number for the first release is 3.2.0. The <build> element has the GXXX, for example, G001. The names of the gzipped tarfiles together with a description of the content follows.

  • fdo-3.2.0_GXXX.tar.gz - source code for Fdo, utilities and third party components
  • fdosdf-3.2.0_GXXX.tar.gz - SDF provider source code and test data
  • fdoshp-3.2.0_GXXX.tar.gz - SHP provider source code and test data
  • fdoarcsde-3.2.0_GXXX.tar.gz - ArcSDE provider source code
  • fdowfs-3.2.0_GXXX.tar.gz - WFS provider source code
  • fdowms-3.2.0_GXXX.tar.gz - WMS provider source code
  • fdordbms-3.2.0_GXXX.tar.gz - source code for the MySQL and ODBC providers

Gunzip the tar.gz file, mv the tar file to the build directory, and tar -xvf the tar file.
