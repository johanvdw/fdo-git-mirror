Environment Variables
 
 

The setenvironment.bat script in the build folder sets the following environment variables required by the build.bat script.

  • FDO
  • FDOMYSQL
  • FDOTHIRDPARTY
  • FDOUTILITIES
  • NLSDIR
  • PATH
  • SDEHOME
  • XALANROOT
  • XERCESCROOT
NoteReview the script to verify that the paths for the .NET Framework, Graphviz, doxygen, and Visual Studio and the values for SDEHOME and FDOMYSQL are correct. You may wish to change some of these values to agree with how you have set up your environment.
NoteThe path to the HTML Help compiler (hhc.exe) does not have to be added to the PATH variable since the full path to the executable is specified in the configuration files used by doxygen to generate the API documentation.

In a cmd.exe window change directory to the build folder and run the setenvironment.bat script.
