Modifying Microsoft Solution or Project Files
 
 

If you decide to modify the solution (.sln) or project (.vcproj) files, be aware that they use path references that contain environment variables defined in the setenvironment.bat script. Launch Developer Studio from a shell in which setenvironment.bat has been run. Then open the solution or project file using a file menu operation.
