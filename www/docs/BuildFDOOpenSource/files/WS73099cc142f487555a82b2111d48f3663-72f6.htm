Subversion Repositories
 
 
 

The SVN client enables you to download files from a Subversion repository into a build folder for the FDO API and a set of supported FDO Providers. A URL identifies the repository and a local path identifies the folder that receives the downloaded files. In this documentation, the build folder is named C:\OpenSource.

To get the source from the repositories, execute the appropriate SVN commands in a cmd.exe window.

NoteThe directory argument can be an absolute or relative path.
NoteThe MySQL and ODBC providers share a repository location under GenericRdbms.
NoteThe fdocore components includes a script called checkoutsvn.bat, which can be used to get updates for the all of the components from the Subversion repositories.
