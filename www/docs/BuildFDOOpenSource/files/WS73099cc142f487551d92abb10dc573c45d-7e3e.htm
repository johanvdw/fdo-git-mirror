Install the Repository Tool
 
 
 

The FDO code is stored in Subversion repositories on the Open Source Geospatial Foundation website. Use a Subversion client to obtain the FDO source from these repositories.To browse the list of available clients, click the Get help? link at the left-hand sideof the https://www.osgeo.org page. A new browser window is displayed; click the About source code version control with Subversion link. Follow the instructions there to obtain and install a client. The instructions in this document are based on the use of the SVN command-line client.
