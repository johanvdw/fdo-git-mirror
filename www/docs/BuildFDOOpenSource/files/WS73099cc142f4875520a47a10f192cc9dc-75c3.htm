Build Scripts
 
 
 

The build folder contains three bat files:

  • build_thirdparty.sh - builds and installs third-party DLLs used by the providers
  • build_parse.sh - recreates the FDO filter and expression language grammar source files from yacc files
  • build_linux.sh - builds and installs the fdo core and provider DLLs
