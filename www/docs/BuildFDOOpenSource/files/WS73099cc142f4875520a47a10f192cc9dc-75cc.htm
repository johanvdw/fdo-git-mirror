Get the Source
 
 
 

The FDO source code is located on the Open Source Geospatial Foundation website. You must first become a registered user of the site and then login before you can download source from the site. Go to https://www.osgeo.org. In the upper right-hand corner there is a link labeled Register. Click on this link and follow the instructions for registering and logging in. Once you have logged in, you may get the source by either downloading a gzipped tar file or doing a “checkout” from a set of Subversion repositories.

NoteThe “checkout” is read-only.
