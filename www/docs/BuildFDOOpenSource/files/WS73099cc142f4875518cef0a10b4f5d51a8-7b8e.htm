Enterprise Edition
 
 
 

FDO is installed as part of the Autodesk Map 3D and Autodesk MapGuide Enterprise installs. The FDO API is not exposed in these products.

Internal Autodesk developers install the SDK on Windows using a Microsoft installer package and on Linux using a gzipped tarfile. The binaries are pre-built. The source code is available in separate gzipped tarfiles.
