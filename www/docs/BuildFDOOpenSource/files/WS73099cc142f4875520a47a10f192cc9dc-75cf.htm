Disk Space
 
 
 

The worse case requirement for build and install disk space occurs if you build both the debug and release versions and install the debug version. In this case you require 2 GB of disk space. This requirement is exclusive of the disk space required for the Subversion client including dependencies (6.1 MB), the build tools (12 MB) and the ArcSDE and MySQL provider clients and ODBC driver (115MB). The final total is 2.13 GB.
