Get Values for the Connection Properties
 
 
 

The types of properties used in connections include:

  • user names
  • passwords
  • file path names
  • URLs
  • symbolic host names
  • service identifiers
  • data store names

Each provider has its own unique set of connection properties. Read abou the connection properties in the chapter about the providers.
