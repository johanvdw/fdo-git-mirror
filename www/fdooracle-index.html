Feature Data Objects Provider for Oracle

The Feature Data Objects (FDO) API provides access to data in a data store. A provider is a specific implementation of the FDO API that provides access to data in a particular data store.

The FDO Provider for Oracle allows access to Oracle Spatial/Locator data in Oracle 10G, Oracle Express Edition (Oracle XE) and Oracle 9i.
