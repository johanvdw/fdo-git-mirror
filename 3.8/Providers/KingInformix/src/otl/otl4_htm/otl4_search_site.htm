Search the OTL site

    Google can narrow down a search to a specific Web site. For exeample, if you want to search the OTL Web site (otl.sourceforge.net) for "stream iterators", it can be done as follows on Google:

         site:otl.sourceforge.net stream iterator

    Miscrosoft Internet Explorer users may download the Google toolbar, which has convenient shortrcuts to do a Web site specific search, and a lot more.
